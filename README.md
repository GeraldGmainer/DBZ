# DBZ Prototype (2015)

The project experiments with core gameplay systems such as movement, flight,
combat mechanics, abilities, and basic character interactions.

## Screenshot
![DBZ Screenshot](dbz.png)

## Playable Characters
- **Son Goku**
- **Future Trunks**

Each character features unique animations, melee attacks, and special abilities.

## Features
- **Third-person movement controller**
- **Flight system** (free movement in 3D space)
- **Melee combat** (light and heavy attacks)
- **Kamehameha ability**
- **Basic hit detection**
- **Health system**
- **Prototype-level environment**

## Purpose of the Project
This prototype was built to explore:
- Real-time combat mechanics  
- Character abilities and FX workflows  
- Animation state machines and transitions  
- Controller responsiveness  
- Gameplay feel and iteration within Unity

## Notes
This is not a full game and was never intended for release.  
It served purely as a learning and experimentation project.
